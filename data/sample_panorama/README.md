# 示例全景数据集

本目录包含用于测试圆柱全景拼接的示例图像序列。

## 目录结构

```
sample_panorama/
├── set1_building/      # 建筑物全景（测试基本拼接）
├── set2_landscape/     # 风景全景（测试大范围拼接）
└── set3_360degree/     # 360度环绕（测试漂移校正）
```

## 数据集要求

### 图像要求
- **重叠度:** 相邻图像需有30%-50%重叠区域
- **格式:** JPEG或PNG
- **EXIF:** 需包含焦距信息（优先）
- **命名:** 按拍摄顺序命名，支持自动排序
  - 推荐格式: `IMG_0001.jpg`, `IMG_0002.jpg`, ...
  - 或: `image_001.jpg`, `image_002.jpg`, ...

### 拍摄建议
1. **使用三脚架（推荐）** 或手持稳定拍摄
2. **固定焦距** - 不要使用变焦
3. **固定曝光** - 使用手动模式或曝光锁定
4. **固定白平衡** - 避免色彩偏移
5. **水平旋转** - 保持相机水平，只做水平旋转

## 数据集说明

### set1_building - 建筑物全景
- **场景:** 建筑外观
- **图像数:** 5-10张
- **覆盖角度:** ~180度
- **用途:** 测试基本圆柱投影和拼接

### set2_landscape - 风景全景
- **场景:** 户外风景
- **图像数:** 8-15张
- **覆盖角度:** ~270度
- **用途:** 测试大范围拼接和特征匹配

### set3_360degree - 360度全景
- **场景:** 室内或开阔场地
- **图像数:** 15-20张
- **覆盖角度:** 360度（首尾重叠）
- **用途:** 测试漂移校正功能

## 使用方法

```bash
# 使用特定数据集运行拼接
python main.py --input data/sample_panorama/set1_building/

# 或在 .env 中配置
IMAGE_PATH=./data/sample_panorama/set1_building
```

## 注意事项

1. **确保图像按顺序命名** - 程序会按文件名排序处理
2. **检查EXIF信息** - 可使用 `exiftool` 或 `identify -verbose` 查看
3. **测试小数据集** - 建议先用5-8张图像测试，确认参数正确后再处理大数据集
